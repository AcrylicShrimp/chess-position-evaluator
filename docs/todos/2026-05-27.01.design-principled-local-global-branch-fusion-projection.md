## Context

The parallel fusion diagnostic showed local and global branch representations are nearly orthogonal by flattened cosine similarity and have different activation scales. The current 1x1 fusion concatenates branches without an explicit contract that they live in compatible feature spaces.

## Why It Is Needed

The user wants a more rigorous approach than naive concatenation: analyze at the formula level why the CNN local branch and attention/global branch differ, then design how to map them into a shared representation space before fusion.

## Completion Criteria

A design document derives the branch representation mismatch, defines a shared-space projection/fusion contract with equations, lists invariants and failure modes, and proposes the next implementation slice with verification criteria.

2. Write the current forward equations for shared, local, global, concat, fuse, and value head paths.
3. Analyze scale, basis, locality/globality, residual, normalization, and covariance mismatch risks.
4. Compare candidate mappings: pre-fusion normalization, per-branch projection, residual aligned fusion, gated fusion, and cross-branch conditioning.
5. Select a minimal controlled design and record equations, invariants, and tests.
