## Context

This TODO tracked the stacked edge-gated attention plus FFN experiment from reviewed plan through implementation, smoke testing, scheduler retuning, full training, and full evaluation.

The original warm-restart run exposed scheduler instability after epoch 30 and was manually stopped. The same stacked architecture was then retuned with a 5-epoch linear warmup plus no-restart cosine decay. That retuned run completed 100 epochs and met the strong practical pass criteria.

The detailed plan and original planned commands remain below for traceability. Final results are recorded in `docs/experiments/2026-05-26.02-ghost-ca-r4-256ch-mhattn4x16-edgegate3-ffn64-mid1-warmupcosine5-lr5e4-100e.md`.

## Why It Is Needed

This experiment isolated attention depth and post-attention per-square channel processing while keeping trunk width, head count, head dimension, dataset split, and most training recipe settings stable. It helped determine that the stacked dynamic attention architecture was viable once the warm-restart scheduler instability was removed.

This experiment is not parameter-matched. Expected `ValueOnlyModel` parameter count is `457,685`, up from `223,573` in the one-layer edge-gate model.

  class NaiveBoardSelfAttention(nn.Module):
      # keep edge-gate attention contract unchanged
      def forward(x: Tensor[B, 256, 8, 8]) -> Tensor[B, 256, 8, 8]

+ class BoardAttentionFFN(nn.Module):
+     def __init__(channels: int, hidden: int):
+         self.net = Sequential(
+             Conv2d(channels, hidden, kernel_size=1, bias=False),
+             BatchNorm2d(hidden),
+             Hardswish(inplace=True),
+             Conv2d(hidden, channels, kernel_size=1, bias=False),
+         )
+     def forward(x):
+         return x + self.net(x)

+ class BoardAttentionBlock(nn.Module):
+     def __init__(channels, heads, head_dim, ffn_hidden):
+         self.attention = NaiveBoardSelfAttention(channels, heads, head_dim)
+         self.ffn = BoardAttentionFFN(channels, ffn_hidden)
+     def forward(x):
+         x = self.attention(x)
+         x = self.ffn(x)
+         return x
