## Goal

Test whether adding zero-initialized chess-geometry relation and distance bias terms to the current best 256-channel, 4-head board attention model improves full validation and test performance.

This experiment is meant to isolate whether the existing attention module benefits from explicit square-to-square geometry priors before moving to heavier graph-attention or typed-edge attention designs.

## Hypothesis Or Expectation

Relation bias may help attention logits separate geometrically meaningful square pairs such as same rank, same file, diagonal, knight move, king adjacency, pawn attack geometry, and distance buckets.

Pass criterion:

- Full validation BCE must be below the current best 256-channel 4-head attention run: `0.607886908`.
- Full test BCE must be below the current best 256-channel 4-head attention run: `0.607948302`.
- If both full-split BCE metrics are worse, treat the relation-bias addition as not helpful under this setup.
- If validation and test disagree, treat the result as inconclusive and prefer replication.

## Key Characteristics

- Architecture variant: `ValueOnlyModel` with `CHANNELS=256`.
- Trunk: six `GhostShuffleBlock` modules with coordinate attention.
- Board attention: one `NaiveBoardSelfAttention` module after the third trunk block.
- Attention layout: `4` heads, `16` dimensions per head, total projection dimension `64`.
- Attention scaling: per-head scale `16 ** -0.5`.
- New geometry priors:
  - `rel_bias`: zero-initialized `[4, 9]` learnable relation bias table.
  - `dist_bias`: zero-initialized `[4, 8]` learnable Chebyshev-distance bias table.
  - Fixed relation masks and distance indices over 64 query/key square pairs.
- Relation names: `same_square`, `same_rank`, `same_file`, `same_diagonal`, `same_anti_diagonal`, `knight_move`, `king_adjacent`, `our_pawn_attack_geometry`, `their_pawn_attack_geometry`.
- Total parameters: `221,465`.
- Geometry-bias parameters: `68`.
- Direct comparison target: `docs/experiments/2026-05-25.04-ghost-ca-r4-256ch-mhattn4x16-mid1-best-recipe-100e.md`.
- Design doc: `docs/designs/2026-05-25.04-attention-relation-bias.md`.

## Verdict Rationale

Verdict is `Pass` because both required full-split BCE gates improved over the current best 256-channel 4-head attention run:

- Validation BCE improved from `0.607886908` to `0.6075506562081671`.
- Test BCE improved from `0.607948302` to `0.6075916815231461`.

The improvement is modest but consistent across validation and test. The tradeoff is that test CP MAE and ECE are slightly worse, so this should be treated as a real BCE improvement rather than a universal metric win.
