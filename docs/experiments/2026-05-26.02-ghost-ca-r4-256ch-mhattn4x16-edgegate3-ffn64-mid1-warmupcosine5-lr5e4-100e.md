## Goal

Test whether the 3-layer edge-gated attention plus FFN model can train stably when the warm-restart schedule is replaced by linear warmup followed by no-restart cosine decay.

This experiment should separate model-capacity signal from the restart instability observed in `ghost-ca-r4-256ch-mhattn4x16-edgegate3-ffn64-mid1-bestrecipe-100e`.

## Hypothesis Or Expectation

The previous stacked-attention run learned faster than nearby attention runs through epoch 30, then destabilized after the scheduler returned to the full `0.0005` peak learning rate.

If the instability was caused by full-strength warm restarts rather than the architecture itself, then a 5-epoch warmup followed by monotonic cosine decay should preserve the early signal and avoid the epoch-30 restart spike.

## Key Characteristics

- Architecture variant: `ValueOnlyModel` with `CHANNELS=256`.
- Attention stack: `3` edge-gated `BoardAttentionBlock` layers with `4` heads, `16` dimensions per head, and FFN hidden width `64`.
- Attention insertion point: after trunk block 3, unchanged.
- Scheduler variable: replace `warm-restart` with `warmup-cosine`.
- Warmup behavior: linearly ramp learning rate from `0.1 * lr` to `lr` across the first `5` epochs.
- Decay behavior: after warmup, cosine decay monotonically approaches `eta_min=1e-6` with no restart.
- Peak learning rate remains `0.0005` to isolate scheduler shape before lowering max LR.
- Expected `ValueOnlyModel` parameters: `457,685`.

## Verdict Rationale

Verdict is `Pass`.

The run satisfies the pre-registered scheduler pass criteria: sampled validation stayed stable through epoch 70, and full validation/test BCE both beat the one-layer edge-gate references.

It also satisfies the pre-registered strong practical pass criteria: full validation BCE `0.6010095617921948` and full test BCE `0.6010760372153482` both beat the static relation-bias references `0.6075506562081671` and `0.6075916815231461`.

There is no calibration warning. Full test ECE `0.0005775951139258682` is better than the one-layer edge-gate warning threshold `0.0017807864956025888` and also better than the static relation-bias test ECE `0.0010109951902087446`.

## Issues And Anomalies

- The 1-epoch smoke uses `--warmup-epochs 1`, so it validates the scheduler mode and target batch path but does not exercise the 5-epoch warmup ramp.
- The validation full-eval report includes `validation_split_is_model_selection_data`, which is expected because the best checkpoint was selected by validation loss. Test metrics are the decisive held-out confirmation.
- Final epoch 100 was slightly worse than the epoch 98 best checkpoint by sampled validation loss. This is expected near convergence and the best checkpoint was retained.

## Follow-Up

- Treat warmup plus no-restart cosine decay as the preferred scheduler for the current stacked attention line unless a later controlled ablation disproves it.
- Inspect edge-gate activations on representative positions to understand what the stacked dynamic attention learned.
- Consider ablations that isolate the contribution of depth versus the FFN path, now that the combined stacked configuration is clearly viable.
