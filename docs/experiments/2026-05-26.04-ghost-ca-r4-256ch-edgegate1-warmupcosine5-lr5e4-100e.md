## Goal

Test whether the strong stacked edge-gated attention result can be explained mostly by the warmup-cosine no-restart scheduler, or whether attention depth plus the FFN path is still required.

This experiment reruns the one-layer dynamic edge-gated attention model with the same warmup-cosine no-restart scheduler that produced the strong stacked edge-gate FFN result.

## Hypothesis Or Expectation

If the scheduler is the dominant factor, then one-layer edge-gated attention with warmup-cosine should move close to the stacked edge-gate FFN warmup-cosine full test BCE `0.6010760372153482`.

If the model remains near the previous one-layer edge-gate result, then the stacked result is primarily explained by added attention depth, the FFN path, increased parameter count, or their combination.

## Key Characteristics

- Architecture variant: `ValueOnlyModel` with `CHANNELS=256` and `model_variant=one-layer-edge-gate`.
- Board mixer: one `NaiveBoardSelfAttention` edge-gated attention layer after trunk block `3`.
- FFN in attention mixer: none.
- Expected `ValueOnlyModel` parameters: `223,573`.
- Scheduler variable: use `warmup-cosine`, matching the successful stacked run.
- Warmup behavior: linearly ramp learning rate from `0.1 * lr` to `lr` across the first `5` epochs.
- Decay behavior: after warmup, cosine decay monotonically approaches `eta_min=1e-6` with no restart.
- Peak learning rate: `0.0005`.
- Dataset and split policy: unchanged.

## Verdict Rationale

Verdict is `Informational` because this is an ablation study with pre-registered interpretation bands rather than a product pass/fail gate.

The full test BCE `0.6061354157950182` satisfies the one-layer scheduler gain band because it improves over the original one-layer edge-gate test BCE `0.6077498676321618` by `0.0016144518371436`, exceeding the `0.001` improvement threshold.

The result does not support the scheduler-only explanation for the stacked run. It remains `0.0050593785796700` worse than the stacked edge-gate FFN warmup-cosine test BCE `0.6010760372153482`, and it barely satisfies the depth/FFN-still-required band by being above `0.6060760372153482`.

Interpretation: warmup-cosine no-restart is a meaningful improvement for shallow one-layer edge-gated attention, but it is not sufficient to reproduce the stacked edge-gate FFN result. The remaining gain is likely attributable to attention depth, the FFN path, added capacity, or their interaction.

- Keep `warmup-cosine` as the scheduler for future attention-family experiments unless a new scheduler ablation is explicitly planned.
- Treat the stacked edge-gate FFN gain as not explained by scheduler alone.
- Future isolation options: one-layer edge-gate plus FFN, three edge-gate layers without FFN, or parameter-matched non-attention control under the same scheduler.
