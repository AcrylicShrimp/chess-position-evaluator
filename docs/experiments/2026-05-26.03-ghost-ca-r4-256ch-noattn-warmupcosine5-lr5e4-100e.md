## Goal

Test whether the latest stacked attention result can be explained by the scheduler change alone.

This experiment reruns the 256-channel non-attention baseline with the same warmup-cosine no-restart scheduler that produced the strong stacked edge-gated attention result.

## Hypothesis Or Expectation

If the scheduler change alone explains most of the improvement, then the no-attention baseline should approach the stacked attention full test BCE `0.6010760372153482`.

If the no-attention model remains much worse, then the stacked result requires either added capacity, attention-specific mixing, or both.

## Key Characteristics

- Architecture variant: `ValueOnlyModel` with `CHANNELS=256` and `model_variant=no-attention`.
- Board mixer: `IdentityBoardAttention`, so the trunk has no full-board self-attention parameters.
- Expected `ValueOnlyModel` parameters: `155,861`.
- Scheduler variable: use `warmup-cosine`, matching the successful stacked run.
- Warmup behavior: linearly ramp learning rate from `0.1 * lr` to `lr` across the first `5` epochs.
- Decay behavior: after warmup, cosine decay monotonically approaches `eta_min=1e-6` with no restart.
- Peak learning rate: `0.0005`.
- Dataset and split policy: unchanged.
