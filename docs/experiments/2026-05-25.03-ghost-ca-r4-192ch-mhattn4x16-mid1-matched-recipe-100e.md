## Goal

Rerun the 192-channel, 4-head board self-attention experiment using the same training recipe as the 256-channel baseline and 256-channel single-head attention runs.

This experiment is meant to determine whether the parameter-matched multi-head attention architecture is competitive when the training recipe is controlled.

## Hypothesis Or Expectation

The model should remain competitive with the 256-channel non-attention baseline when trained with the same recipe. Beating the prior 256-channel single-head attention run would be a stronger positive signal, but is not required for this parameter-matched test to remain useful.

## Key Characteristics

- Architecture variant: `ValueOnlyModel` with `CHANNELS=192`.
- Trunk: six `GhostShuffleBlock` modules with coordinate attention.
- Board attention: one `NaiveBoardSelfAttention` module after the third trunk block.
- Attention layout: `4` heads, `16` dimensions per head, total projection dimension `64`.
- Attention scaling: per-head scale `16 ** -0.5`.
- Total parameters: `158,965`.
- Attention parameters: `49,152`.
- Direct predecessor run: `docs/experiments/2026-05-25.02-ghost-ca-r4-192ch-mhattn4x16-mid1-100e.md`.
- This run explicitly corrects the predecessor run's recipe mismatch.

## Verdict Rationale

Verdict is `Pass` for the primary target because the matched-recipe 192-channel, 4-head attention model beat the 256-channel non-attention baseline on both full validation and full test BCE.

The result confirms that the predecessor run's poor metrics were caused at least in large part by the training recipe mismatch. With the controlled recipe, the model improved full test BCE from the baseline's `0.610118656` to `0.609783332`, improved test CP MAE by about `1.97cp`, and improved test ECE from `0.002717377` to `0.001261373`.

The result does not beat the prior 256-channel single-head attention run. The matched 192-channel 4-head model is worse by `0.001267300` test BCE and about `3.82cp` test CP MAE. Treat it as a viable lightweight parameter-matched variant, not as the current highest-accuracy architecture.

## Issues And Anomalies

- The predecessor run remains preserved as a completed but non-comparable experiment because it used mismatched batch size, step count, learning rate, and sample exposure.
- The validation full-evaluation report includes `validation_split_is_model_selection_data`; the test report has no warnings.
- Sampled validation loss and full-split BCE are close but not identical, so final conclusions use full-split evaluation.

## Follow-Up

- Keep this run as evidence that the parameter-matched 192-channel 4-head attention model is baseline-competitive when trained with the correct recipe.
- Do not treat this as a replacement for the 256-channel single-head attention model when optimizing for top accuracy.
- Consider a 256-channel 4-head attention run to isolate single-head versus multi-head at fixed trunk width.
- Consider a 192-channel no-attention control if parameter matching attribution remains important.
