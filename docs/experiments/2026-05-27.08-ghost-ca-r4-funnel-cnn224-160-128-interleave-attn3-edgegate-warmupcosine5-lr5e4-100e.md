## Goal

Train and evaluate a funnel CNN plus interleaved narrow attention model.

This experiment tests whether CNN refresh blocks between narrow attention blocks reduce the late-stage spectral concentration observed in the previous `funnel-cnn224-160-128-attn6-edgegate` run while preserving the speed advantage of narrow attention.

## Hypothesis Or Expectation

The model may improve over the previous six-attention funnel by alternating global relation mixing with local/spatial refresh. The expected win condition is not only lower BCE, but also healthier post-training activation-rank behavior near the final trunk layers.

The main risk is that reducing attention depth from six to three makes the model too small or under-expressive despite the CNN refresh blocks.

## Key Characteristics

- Model variant: `funnel-cnn224-160-128-interleave-attn3-edgegate`.
- Architecture: `24 -> 224`, two `C224` Ghost blocks, `224 -> 160`, two `C160` Ghost blocks, `160 -> 128`, two `C128` Ghost blocks, then `[C128 q+k edge-gated attention block -> C128 Ghost block] x3`.
- Attention: `4` heads, `16` head dim, FFN hidden size `64`.
- Value head: material-difference scalar semantics matching the stacked baseline and previous funnel experiment.
- Distinct from previous funnel: replaces `Attention C128 x6` with three attention blocks interleaved with CNN refresh.
- Design doc: `docs/designs/2026-05-27.02-funnel-interleaved-attention.md`.

## Verdict Rationale

Fail by the pre-registered BCE criteria. Full validation BCE regressed by `+0.0018568900784445` versus the previous six-attention funnel and by `+0.0030354453208766` versus the cleaned stacked baseline. Full test BCE regressed by `+0.0018743843026339` versus the previous six-attention funnel and by `+0.0031092693744155` versus the cleaned stacked baseline.

The secondary architecture signal is mixed but useful. ECE improved on both validation and test, and rank analysis shows that interleaving CNN refresh blocks reduced the previous final-attention spectral concentration. The result suggests the refresh idea is structurally promising, but this specific `C128` three-attention configuration is underpowered.

## Issues And Anomalies

- No training instability, NaN, or validation blow-up was observed.
- The model remained consistently behind stronger references throughout training despite stable convergence.
- The best checkpoint was epoch `100`, so the run did not show early overfitting.
- The calibration improvement should not be treated as an overall win because BCE and CP-equivalent MAE regressed materially.

## Follow-Up

- Do not promote `funnel-cnn224-160-128-interleave-attn3-edgegate` as a new baseline.
- Keep the rank result as evidence that CNN refresh blocks can reduce late-stage attention spectral concentration.
- If this idea is revisited, preserve more attention depth or capacity while retaining the refresh mechanism.
- Consider a follow-up that alternates more attention blocks with lightweight refresh blocks, rather than reducing attention depth to three.
