## Procedure

1. Regenerated processed datasets after the split-policy change:

```bash
cargo run --release -p preprocess
```

2. Trained the baseline checkpoint:

```bash
PYTHONUNBUFFERED=1 uv run cpe train ghost-ca-r4-256ch-split90505-baseline-100e \
  --epochs 100 \
  --steps 1024 \
  --batch 2048 \
  --lr 0.0005 \
  --wd 0.0001 \
  --t0 10 \
  --t-mult 2 \
  --eta-min 1e-6 \
  --grad-clip 5
```

3. Strict-loaded the best checkpoint into `ValueOnlyModel`; no missing or unexpected state dict keys were reported.

4. Evaluated the best checkpoint on the full validation split:

```bash
PYTHONUNBUFFERED=1 uv run cpe eval-dataset ghost-ca-r4-256ch-split90505-baseline-100e-best --split validation --full --batch 4096 --device auto
```

5. Evaluated the best checkpoint on the full test split:

```bash
PYTHONUNBUFFERED=1 uv run cpe eval-dataset ghost-ca-r4-256ch-split90505-baseline-100e-best --split test --full --batch 4096 --device auto
```
