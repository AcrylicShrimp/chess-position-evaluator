# Tree Search

This crate is a dormant prototype.

It was started as a Rust-side game-tree search layer for SEE-based static or
quiescence search that could eventually use ONNX model evaluation. It is not
part of the active training, evaluation, inference, or gameplay workflow.

Do not treat the current crate as production search code. Before reviving it,
define the search contract, model-evaluation boundary, move ordering rules, SEE
semantics, and verification plan.
